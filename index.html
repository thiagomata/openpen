    Openpen

    We are working on some easy way to convert 2D drawing into objects. In this purpose, this module needs to be able to resize and rotate the elements and compare them with the known templates.

    Instalation

    There is not some easy installation process yet. Sorry :( But, you still can just copy our javascript :D

    Demonstration

    In the current version, we are able to resize any matrix, estimate the missing values when it grows and merge the values when reduce.
